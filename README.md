# Offra Docs

Public documentation website for the Offra iOS app.

## Pages

- Home: `index.html`
- Privacy Policy: `privacy.html`
- Support: `support.html`
- Terms of Use: `terms.html`

## GitHub Pages setup

1. Open repository settings.
2. Go to Pages.
3. Select "Deploy from a branch".
4. Select branch `main`.
5. Select folder `/root`.
6. Save.

Example URLs after deployment:

- `https://YOUR_GITHUB_USERNAME.github.io/offra-docs/`
- `https://YOUR_GITHUB_USERNAME.github.io/offra-docs/privacy.html`
- `https://YOUR_GITHUB_USERNAME.github.io/offra-docs/support.html`
